CRUDTASK - Academic Task Management System
Project Description
CRUDTASK is a complete web application for managing academic tasks. The system allows users to register, log in, and manage their tasks, while administrators can monitor all system activity. The application uses a fake API with JSON Server to simulate a real backend.
Main Features
Authentication System
New user registration (automatic role: user)
Login with credential validation
Session persistence using LocalStorage
Secure logout
Role Management
User: Manages their own tasks
Admin: Monitors the entire system and manages all tasks
Role-based route protection
Automatic redirection based on permissions
Task Management
Full CRUD: Create, Read, Update, Delete tasks
Advanced filters: By category, priority, status, and date
Search: By title, ID, or tags
Status change: Pending → In Progress → Completed
Assignment: Tasks assigned to specific users
Dashboard and Metrics
Admin Dashboard: System overview with statistics
User Dashboard: Personalized view of academic progress
Real-time metrics: Total tasks, completed, pending
Progress charts: Visual performance tracking
User Profiles
Editable personal information
Individual statistics
Activity history
Account settings
